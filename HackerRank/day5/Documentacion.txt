Objetivo

En este desafío, usaremos bucles para realizar algunos cálculos matemáticos. Revisa la pestaña Tutorial para aprender más.

Tarea

Dado un número entero, n, imprime sus primeros 10 múltiplos. Cada múltiplo n × i (donde 1 ≤ i ≤ 10) debe imprimirse en una nueva línea con el formato:

n x i = resultado

Ejemplo

La salida debería verse así:

3 x 1 = 3
3 x 2 = 6
3 x 3 = 9
3 x 4 = 12
3 x 5 = 15
3 x 6 = 18
3 x 7 = 21
3 x 8 = 24
3 x 9 = 27
3 x 10 = 30

Formato de entrada

Un único número entero, n.

Restricciones

Formato de salida

Imprime 10 líneas de salida; cada línea i (donde 1 ≤ i ≤ 10) contiene el múltiplo de n con el formato:

n x i = resultado

Entrada de ejemplo

2

Salida de ejemplo

2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
2 x 4 = 8
2 x 5 = 10
2 x 6 = 12
2 x 7 = 14
2 x 8 = 16
2 x 9 = 18
